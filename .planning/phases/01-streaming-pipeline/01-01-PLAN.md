  <action>
    Implement the Claude Code subprocess bridge that spawns the CLI and yields text chunks from the NDJSON stream.

    **src/claude_teletype/bridge.py** must contain:

    1. `async def stream_claude_response(prompt: str) -> AsyncIterator[str]`:
       - Spawns `claude -p {prompt} --output-format stream-json --verbose --include-partial-messages` using `asyncio.create_subprocess_exec()`
       - Pipes stdout and stderr (stdout=asyncio.subprocess.PIPE, stderr=asyncio.subprocess.PIPE)
       - Reads stdout line by line with `proc.stdout.readline()`
       - Parses each line with `json.loads()`
       - Skips lines that fail json.loads (JSONDecodeError)
       - Filters for: msg["type"] == "stream_event" AND msg["event"]["type"] == "content_block_delta" AND msg["event"]["delta"]["type"] == "text_delta"
       - Yields msg["event"]["delta"]["text"] when non-empty
       - On EOF (empty line from readline), breaks the loop
       - Calls `await proc.wait()` after the loop
       - Wraps the entire function in try/finally to ensure proc.terminate() + proc.wait() on exceptions/interruption

    2. `def parse_text_delta(line: bytes) -> str | None`:
       - A helper function (non-async) that takes a raw line (bytes) and returns the text_delta string or None
       - This enables unit testing of the NDJSON parsing without a subprocess
       - Decodes bytes to str, json.loads, applies the same filtering as above

    **tests/test_bridge.py** must contain:

    1. Test parse_text_delta with a valid text_delta NDJSON line returns the text
    2. Test parse_text_delta with message_start event returns None
    3. Test parse_text_delta with content_block_start event returns None
    4. Test parse_text_delta with content_block_delta but input_json_delta (tool use) returns None
    5. Test parse_text_delta with content_block_stop returns None
    6. Test parse_text_delta with message_stop returns None
    7. Test parse_text_delta with result message returns None
    8. Test parse_text_delta with malformed JSON returns None
    9. Test parse_text_delta with empty line returns None
    10. Test stream_claude_response yields correct text from a mock subprocess (use unittest.mock.patch on asyncio.create_subprocess_exec to provide a mock process with pre-loaded NDJSON lines on stdout)

    Use the exact NDJSON message formats from the research document (01-RESEARCH.md Code Examples section) as test fixtures.
