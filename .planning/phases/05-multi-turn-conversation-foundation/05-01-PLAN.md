<feature>
  <name>Bridge multi-turn session support</name>
  <files>src/claude_teletype/bridge.py, tests/test_bridge.py</files>
  <behavior>
    **StreamResult dataclass:**
    - Fields: session_id (str|None), is_error (bool), error_message (str|None), cost_usd (float|None), model (str|None), num_turns (int|None), usage (dict|None), model_usage (dict|None)

    **parse_session_id(line: bytes) -> str | None:**
    - Input: NDJSON system/init line -> returns session_id string
    - Input: Any other NDJSON line -> returns None
    - Input: Malformed JSON -> returns None
    - Input: Empty/whitespace -> returns None

    **parse_result(line: bytes) -> dict | None:**
    - Input: NDJSON result line -> returns dict with is_error, result, cost_usd, num_turns, session_id, usage, model_usage
    - Input: Any other NDJSON line -> returns None
    - Input: Malformed JSON -> returns None

    **calc_context_pct(model_usage: dict | None) -> str:**
    - Input: {"claude-sonnet-4-5-20250929": {"inputTokens": 9, "outputTokens": 143, "cacheReadInputTokens": 39900, "cacheCreationInputTokens": 439, "contextWindow": 200000}} -> "20%"
    - Input: None -> "--"
    - Input: {} -> "--"
    - Input: model with contextWindow: 0 -> "--"

    **extract_model_name(model_usage: dict | None) -> str | None:**
    - Input: {"claude-sonnet-4-5-20250929": {...}} -> "claude-sonnet-4-5-20250929"
    - Input: None -> None
    - Input: {} -> None

    **stream_claude_response(prompt, session_id=None, proc_holder=None) -> AsyncIterator[str | StreamResult]:**
    - When session_id is None: does NOT include --resume in args
    - When session_id is provided: includes --resume session_id in args
    - When proc_holder is provided: appends proc to proc_holder after spawn
    - Yields text chunks (str) during streaming
    - Yields a single StreamResult as the FINAL item with session metadata parsed from system/init and result NDJSON messages
    - On exception: terminates process, raises (same as before)
  </behavior>
