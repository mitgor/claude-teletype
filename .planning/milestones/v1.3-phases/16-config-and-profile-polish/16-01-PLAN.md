  <name>Task 2: Annotate config show output with source tags and tests</name>
  <files>src/claude_teletype/config.py, src/claude_teletype/cli.py, tests/test_config.py</files>
  <action>
In `config.py`:
1. Add a new function `def resolve_sources(config_path: Path | None = None) -> dict[str, str]` that determines the source of each effective config value. The function should:
   - Load defaults from `TeletypeConfig()` (all fields).
   - Load file config from TOML (if file exists) using `load_config(config_path)`.
   - Check env vars for each field using `CLAUDE_TELETYPE_{FIELD_NAME.upper()}` pattern.
   - For each field in TeletypeConfig (excluding `custom_profiles`), determine the source:
     - If env var `CLAUDE_TELETYPE_{name.upper()}` is set in `os.environ` -> source = `"env (CLAUDE_TELETYPE_{name.upper()})"`
     - Else if file config has a different value from defaults -> source = `"file ({config_path})"` (use the actual path)
     - Else -> source = `"default"`
   - Return `{field_name: source_string}` dict.
   - Note: CLI flag source detection is intentionally OUT OF SCOPE. The `show()` subcommand runs via `claude-teletype config show` — a separate Typer subcommand that does not receive the main command's CLI flags (--delay, --backend, etc). CLI flags are only meaningful during `main()` execution. Users already know which CLI flags they typed, so `config show` focuses on the three layers they CAN'T easily introspect: defaults, file overrides, and env var overrides.

In `cli.py`:
1. Import `resolve_sources` from config module.
2. Rewrite the `show()` command to:
   - Call `resolve_sources()` to get the source mapping.
   - For each config field, print in format: `{key} = {value}  # {source}` — use two spaces before `#` for readability.
   - API keys still show `***` for non-empty values instead of the actual key.
   - Group output with section headers: `[general]`, `[printer]`, `[llm]`, `[keys]`.
   - Keep the existing header lines (`Config file: ...` and `File loaded: ...`).

Example output format:
```
Config file: /Users/mit/Library/Application Support/claude-teletype/config.toml
File loaded: True

[general]
delay = 75.0  # default
no_audio = False  # env (CLAUDE_TELETYPE_NO_AUDIO)
no_tui = False  # default
transcript_dir = transcripts  # file (/Users/.../config.toml)

[printer]
device = None  # default
printer_profile = juki  # file (/Users/.../config.toml)

[llm]
backend = claude-cli  # default
model =   # default
system_prompt = ''  # default

[keys]
openai_api_key =   # default
openrouter_api_key =   # default
```
